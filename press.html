01 About Cycl

We make bottle and can returns easier to place, use, and measure.

Cycl builds mobile first return infrastructure for bottles and cans. The platform turns simple return collectors into connected stations with AI recognition, partner rewards, and impact reporting.

Our goal is to make return points fit into everyday places, from malls and campuses to libraries, venues, offices, and public locations. More convenient return points can help more containers make it back into useful material streams.
